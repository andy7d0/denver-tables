подсистемы

у терапвета есть своя "область" на сервере
там его клиенты
для них опросы
идентифицируется адресом, защищен пином
на сервере область задается хешем адреса (сервер не знает настоящего адреса!)
внутри области все шифровано пином
есть индекс клиентов
клиент - папка
есть у нее индекс опросов + метаданные клиента
и данные опросов

отдельно есть клиентские ссылки на опросы (они зашифрованы ключами опроса)

1) создание опроса
	кто создает (что-то вроде логина)
	для кого создает (пациент)
	стадия (шаг таблицы)
	версия (если надо повторить)
	куда отвечать
	+ данные
	- сообщение
	- фильт уровней

	публикация - получение ссылки
	ссылка:
	адрес (кто создал, для кого, стадия, версия)
	ключ расшифровки - адрес + пин терапевта
